# AI CRM Interaction Logger

AI-powered CRM application for logging HCP interactions.

## Features

- AI interaction logging
- Edit existing interaction
- Dynamic summarization
- Voice input
- Sentiment analysis
- Materials & sample tracking

## Tech Stack

Frontend:
- React
- Axios

Backend:
- FastAPI
- LangGraph
- LangChain
- Groq

## Run Frontend

```bash
cd frontend
npm install
npm run dev
```

## Run Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

Open:

http://localhost:5173
